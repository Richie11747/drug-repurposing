# drug-repurposing
Autonomous Claude agent pipeline that rediscovers known anti-Chagas drug hits (HIV protease inhibitors) and screens FDA-approved drugs for novel repurposing candidates against cruzain, the validated drug target of Trypanosoma cruzi. Built for Built with Claude: Life Sciences.
Molecular Detective is an agentic drug-repurposing pipeline for Chagas disease — a WHO-listed neglected tropical disease affecting 6–8 million people, still treated with two 40-year-old, poorly tolerated drugs. The pipeline screens the FDA-approved drug library against cruzain, T. cruzi's validated cysteine-protease target, using Boltz-2 binding predictions. Before trusting any novel result, it has to pass a sanity check: correctly rediscovering known repurposing hits (nelfinavir, lopinavir, ritonavir, saquinavir) from the published literature. Only then does it surface new candidates — with full reproducibility, since every score is traced back to the exact structure, model, and code that produced it.
